## Project Objective
Build a centralized AI chatbot platform capable of serving multiple internal enterprise products while maintaining secure product isolation. The platform extracts content from uploaded documents, cleans it using an LLM, converts it into structured Markdown, and prepares it for semantic search using Qdrant.

## Modules
1. Authentication & Product Registry
2. Document Upload Pipeline
3. PDF/DOCX/TXT Text Extraction
4. LLM-based Markdown Sanitization
5. Vector Embedding & Qdrant Storage
6. Enterprise Chat Interface
