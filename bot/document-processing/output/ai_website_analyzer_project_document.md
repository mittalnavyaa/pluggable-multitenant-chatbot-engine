# AI Website Analyzer Platform
## Project Overview
The AI Website Analyzer Platform helps organizations evaluate website quality using SEO, GEO, AEO, accessibility, and performance metrics. The platform crawls websites, extracts metadata, analyzes content quality, and generates optimization recommendations.

## Functional Requirements
The system should accept website URLs, crawl multiple pages, extract metadata, evaluate page performance, detect SEO issues, and generate AI-powered improvement suggestions.
