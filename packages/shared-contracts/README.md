# @envoy/shared-contracts

Authoritative API contract definitions, typescript interfaces, JSON schemas, and payload validator helpers shared between Envoy's frontend components, backend proxy SDKs, and centralized API gateways.

---

## Folder Structure

- `src/types/index.ts` — Authoritative TypeScript type definitions for network interactions.
- `src/validation/index.ts` — Lightweight payload validation rules.
- `API_SPECIFICATION.md` — Detailed API reference manual.
